# 쇼츠 5개 자동 제작

긴 영상 → **혼자 봐도 말이 되는 1분 쇼츠 5개**(9:16). 단순 하이라이트 1초가 아니라 설정→훅→결론이 담긴 자체 완결 클립.

## 입력
- 원본 영상, `output/<base>_transcript.md`(읽기 좋은 전사), `output/_workspace/10_research.md`(하이라이트·hook 점수)
- 전사/리서치 없으면 먼저 `transcript_export.py` / 리서처 실행.

## 절차

### 1. 5개 선정 (서로 다른 맥락)
리서치 하이라이트(hook 4점+) 중 **다른 각도** 5개: 인트로훅 / 핵심논리 / 반전 / 구체사례 / 결론. 같은 주제 반복 금지.

### 2. 1분 윈도우로 확장 (맥락 완결)
각 하이라이트를 45~75초로 확장:
- **시작점** — 훅 문장이 클립 **첫 3초**에 오게. 그 앞에 꼭 필요한 한 문장만 설정으로.
- **끝점** — 결론/임팩트 문장에서. 문장 중간에서 끊지 않기.
- 너무 길면(>75초) 곁가지를 빼서 압축, 너무 짧으면(<40초) 맥락을 더 포함.

### 3. 타임코드 = 원본 시각
`make_shorts.py`는 **원본 영상**을 자른다. 리서치에 [원본]/[컷] 두 기준이 있으면 **[원본]** in-out을 쓴다.

### 4. 제작
```bash
python3 engine/make_shorts.py "원본.mp4" "12:25-13:20" "23:10-24:05" ...
```
→ `output/shorts/short_01~05.mp4` (9:16 1080x1920, -14 LUFS). 자막은 미포함(별도).

### 5. 메모 → `output/_workspace/50_shorts.md`
```
## 쇼츠 1 — "유튜브는 조회수가 아니라 영업사원"
- in-out(원본): 03:35-04:30 (55초)
- 첫 3초 훅: "유튜브를 조회수로만 보면 망합니다"
- 선정 이유: 채널 핵심 철학, 제목과 직결
```

## 원칙 (왜)
- **첫 3초가 쇼츠의 생사** — 시작이 약하면 즉시 이탈. 가장 강한 단언/숫자/궁금증으로 연다.
- **자체 완결 > 1분 정확히** — 맥락이 끊기면 1분이어도 실패. 45~75초 안에서 완결점을 찾는다.
- **5개 다양성** — 한 영상에서 다른 사람에게 꽂힐 5개의 다른 진입점.
